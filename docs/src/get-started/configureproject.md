## 烧录准备
1. 进入`固件烧录`页面。
![](images/image2.png)
2. 在`芯片选择`选项框中选择你使用的**芯片型号**和**存储器类型**
3. 在`接口设置`选项框中选择要烧录的硬件对应的**串口号**和**波特率**
4. 点击`连接设备`，等待片刻，设备将连接，如图所示。
![](images/image3.png)
5. 此时，将编译产物拖入`选择固件文件`框中，此处以SF32LB52为例。
   - .bin格式（需要自行填入地址），并手动在烧录地址框中修改地址为寄存器对应地址。
   ![alt text](images/image5.png)
   - .hex/.elf/.axf文件，直接拖入框中，自动填入地址
   ![alt text](images/image6.png)
   - sftool_param.json，自动填入bootloader、ftab、main的二进制文件及其地址，最为便捷，推荐使用。
6. 确认以上配置无误后，点击`开始烧录`按钮，烧录过程中，正在烧录的文件将高亮显示
   ![alt text](images/image7.png)
7. 烧录完成后，系统日志框将显示`烧录完成`，此时断开连接，拔下并重新插入设备，烧录步骤结束
