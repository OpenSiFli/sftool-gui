## 获取最新版本的sftool-gui
1. [点此](https://github.com/OpenSiFli/sftool-gui/releases)进入sftool-gui的releases，注意有Latest标识为最新版本。
   ![alt text](images/image1.png)
2. 在如图的Assets中，选择适合你系统的版本。如果你想跳过安装步骤直接使用，可以下载带portable标记的版本。
3. 点击文件名即可进行下载

## 安装 sftool-gui
1. 如果你下载了带有portable标记的版本，你将得到一个带有程序本体的压缩包，解压到任意路径运行即可。

2. 如果你下载了常规版本，请像安装其他程序那样安装sftool-gui，并运行它。
